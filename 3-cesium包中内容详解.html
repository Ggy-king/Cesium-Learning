• Apps: Cesium详细的范例程序(Apps\Sandcastle\gallery示例demo)
    • { CesiumViewer: 一个简单的Cesium初始化代码 SampleData: 相应示例的数据 Sandcastle: 相应的沙盒 所有的示例 TimelineDemo: 时间轴示例 }
  • Build: Release包,打包后的库文件(压缩和未压缩)、sdk文档
    • { Cesium: 压缩后的cesium库文件 CesiumUnminified: 未压缩的cesium库文件 Documentation: 所有API的参考文档 }
  • Source: 所有源码
  • Specs: cesium自动化单元测试,采用jasmine框架,可以实现自动化单元测试以及接口覆盖率等统计效果
  • ThirdParty: 外部依赖库,不同于cesium的第三方库
  • 其他文件
    • { index.html: Web导航首页 server.js: 基于node.js的web服务应用 gulpfile.js: 打包脚本 }
